51. YouTube Tips & Troubleshooting

51. YOUTUBE TIPS

PROBLEM YT INVITE & CARA FIX NYA

Sepertinya Anda tidak berada di negara yang sama

Cara fix nya :

  • Klik link ini https://payments.google.com/gp/w/u/0/home/settings
  • Scroll ke bawah ada tulisan "tutup profil pembayaran" klik masukan password
  • Terus alasannya bebas pilih yg mana aja, abis itu klik "lanjutkan"
  • Profil berhasil tertutup klo ada tulisan "tambahkan metode pembayaran" di pay.google.com
  • Close tab abis itu tinggal join ulang ke link invitenya.

Anda sudah berada di grup keluarga (maksimal setahun 1x ganti)

Cara fix nya :

  • Klik link ini https://families.google.com/families
  • Klik 3 baris di kiri atas / gambar profil
  • Klik "Keluar dari grup keluarga"
  • Setelah keluar, tinggal join ulang ke link invitenya.
